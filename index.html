Engine Architecture

How PySpector translates source structures into dynamic taint propagation:

01 AST Generation Python CLI generates native syntax AST structures, serializing data targets into normalized JSON arrays.
02 Call Graph Map Rust Core constructs project-wide maps of function boundaries, resolving cross-file reference paths.
03 CFG Compilation Calculates basic blocks mapping branch paths, conditional evaluations, and loops control flows.
04 Taint Worklist Executes fix-point worklist algorithms matching source injections to unsafe code destinations.

Quick Start CLI

PySpector is distributed via PyPI. Get started with native command lines in seconds: 

# Install PySpector globally or inside virtual environment
pip install pyspector

# Verify local Rust compiler is configured if compiling from source
cargo --version
# Scan active local directory
pyspector scan ./my_python_project

# Scan specifying JSON output file
pyspector scan ./project -f json -o scan_report.json

# Scan enabling interactive setup wizard
pyspector scan --wizard

# Execute scans including LLM security rules
pyspector scan ./llm_agents --ai
# Export to standard SARIF files for Github Code Scanning integration
pyspector scan ./src -f sarif -o report.sarif

# Configure pre-commit hook layers
./scripts/setup_hooks.sh

# Generate automated scheduled scans
./scripts/setup_cron.sh
Plugin System

PySpector handles post-scan execution layers, letting developers orchestrate warnings or generate custom security artifacts. 

# List discovered local plugin states
pyspector plugin list

# Audit, calculate check-hashes and mark a plugin as secure
pyspector plugin trust plugin_name

# Install external pipeline hooks
pyspector plugin install ./path/to/hook.py --trust

# Run scans linking active trusted post-hooks
pyspector scan ./target --plugin aipocgen --plugin-config ./config.json

from pyspector.plugin_system import PySpectorPlugin, PluginMetadata

class MyPlugin(PySpectorPlugin):
    @property
    def metadata(self) -> PluginMetadata:
        return PluginMetadata(
            name="high_summary",
            version="0.1.0",
            author="Auditor",
            description="Exports summary of findings"
        )
    
    def process_findings(self, findings, scan_path, **kwargs):
        highs = [f for f in findings if f.get("severity") == "HIGH"]
        return {"success": True, "message": f"Processed {len(highs)} issues"}
